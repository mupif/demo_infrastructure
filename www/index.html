MuPIF Demo Infrastructure

MUPIF Demo Infrastructure is a containerized version of the MuPIF platform components, packaged to run within a single container. It is intended solely for demonstration and training purposes. For production-grade deployments, it is highly recommended to adopt a distributed setup secured within a private VPN network.
